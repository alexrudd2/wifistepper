{% mdicon information %}About the Wi-Fi Stepper

Made With {% mdicon heart, .inline, #f00 %} By Good Robotics in Santa Cruz, CA

Sponsored by Crowd Supply and You, our backers. Thank you.

We started this project because we were finding it difficult to drive stepper motors simply and cheaply. We saw an opportunity to help our fellow makers and build a board that incorporates the logic-controller, motor driver, H-Bridges, and provides secure access over Wi-Fi. This all-in-one board can be easily integrated into projects like robotic arms, pumps, CNC axis additions, 3D printers, linear camera sliders, automatic pet feeders, security gate openers, and any other project that involves stepper motors.

For more information and updates to this project, please visit wifistepper.com
